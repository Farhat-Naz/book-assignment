## 🎯 Overview

This repository contains a complete Docusaurus-based educational platform for learning **Physical AI & Humanoid Robotics**. The course provides:

- **8 Modules** covering foundations to advanced topics
- **36 Chapters** with comprehensive theoretical content
- **36 Hands-on Labs** with runnable code examples
- **180 Hours** of learning content
- **3 Capstone Projects** for real-world application

## 📚 Course Structure

### Modules

1. **Foundations of Physical AI** - Introduction, math, robot anatomy, ROS2
2. **Robot Kinematics** - Forward/inverse kinematics, motion planning
3. **Dynamics & Control** - Rigid body dynamics, control theory
4. **Perception** - Computer vision, SLAM, sensor fusion
5. **Bipedal Locomotion** - Walking, balance, terrain adaptation
6. **AI & Machine Learning** - RL, deep learning, imitation learning
7. **Human-Robot Interaction** - Speech, gestures, collaboration
8. **Integration & Capstone** - System architecture, ethics, projects

## 🚀 Quick Start

### Prerequisites

- Node.js 20.0 or higher
- npm or yarn
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR-USERNAME/Book-Assignment.git
cd Book-Assignment/humaniod-robotics

# Install dependencies
npm install

# Start development server
npm start
```

The site will open at `http://localhost:3000`

### Build for Production

```bash
npm run build
npm run serve
```
